## Firewall Rules
# Пример правил брандмауэра для ограничения доступа
# Разрешить доступ с VLAN 10 к VLAN 20
iptables -A FORWARD -i eth0.10 -o eth0.20 -j ACCEPT
# Запретить доступ с VLAN 20 к VLAN 10
iptables -A FORWARD -i eth0.20 -o eth0.10 -j DROP

# Примечание: Правила брандмауэра могут быть настроены через консоль управления облака, например, AWS Security Groups или Azure Network Security Groups.
