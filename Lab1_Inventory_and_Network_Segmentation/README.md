# Лабораторная работа 1: Инвентаризация и сетевое сегментирование

## Цель
Выполнить инвентаризацию существующих ресурсов в облачной инфраструктуре и настроить сетевое сегментирование.

## Задачи
1. Использовать Ansible для автоматизации инвентаризации серверов в облаке.
2. Настроить сетевое сегментирование для облачных ресурсов.
3. Ограничить доступ между сегментами с помощью правил брандмауэра.

## Описание
В этой лабораторной работе студенты будут использовать Ansible для создания инвентаризации облачных серверов на Windows и Ubuntu, а также настройки сетевого сегментирования. Студенты также научатся применять правила брандмауэра для ограничения доступа между различными частями облачной сети.

## Ресурсы
-   Ansible
-   SSH доступ к Ubuntu серверам
-   RDP доступ к Windows серверам
-   Документация по VLAN и облачной инфраструктуре
