# Лабораторная работа 4: Настройка межсетевого экрана и VPN

## Цель
Настроить межсетевой экран и VPN для безопасного доступа к ресурсам.

## Задачи
1. Настроить iptables или UFW для управления входящим и исходящим трафиком.
2. Установить и настроить OpenVPN для безопасного доступа к ресурсам.
3. Провести тестирование доступа через VPN.

## Описание
В этой лабораторной работе студенты будут настраивать межсетевой экран с использованием iptables или UFW и устанавливать OpenVPN в Docker для обеспечения безопасного доступа к удаленным серверам. Студенты научатся управлять трафиком и проверять доступ через VPN.

## Ресурсы
-    Docker
-    OpenVPN
-    iptables или UFW
-    SSH доступ к серверам
-    Файл инвентаризации Ansible из первой лабораторной работы
-    Файл переменных `vars.yml`, содержащий логины и пароли для доступа к серверам

## Подготовка
1. Убедитесь, что все серверы доступны и настроены.
2. Убедитесь, что файлы `inventory.yml` и `vars.yml` правильно настроены и содержат актуальные данные.
3. Установите Docker на свой локальный компьютер, если он еще не установлен. Инструкции можно найти на [официальном сайте Docker](https://docs.docker.com/get-docker/).
