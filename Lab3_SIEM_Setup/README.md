# Лабораторная работа 3: Настройка Wazuh

## Цель
Настроить систему управления безопасностью и мониторинга с помощью Wazuh, установив его в Docker на своем локальном компьютере.

## Задачи
1. Установить и настроить Docker на своем локальном компьютере, если он еще не установлен.
2. Запустить Wazuh в Docker для сбора и анализа логов с удаленных серверов.
3. Установить Grafana и настроить дашборды для визуализации данных Wazuh.

## Описание
В этой лабораторной работе студенты будут устанавливать Wazuh в Docker на своих локальных компьютерах и настраивать его для сбора и анализа логов с удаленных серверов. Grafana будет использоваться для визуализации данных о безопасности. Студенты научатся создавать дашборды, отображающие состояние системы и инциденты безопасности.

## Ресурсы
-     Docker
-     Wazuh
-     Grafana
-     SSH доступ к серверам
-     Файл инвентаризации Ansible из первой лабораторной работы
-     Файл переменных `vars.yml`, содержащий логины и пароли для доступа к серверам

## Подготовка
1. Убедитесь, что все серверы доступны и настроены.
2. Убедитесь, что файлы `inventory.yml` и `vars.yml` правильно настроены и содержат актуальные данные.
3. Установите Docker на свой локальный компьютер, если он еще не установлен. Инструкции можно найти на [официальном сайте Docker](https://docs.docker.com/get-docker/).
