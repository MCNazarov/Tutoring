# Tutoring
# Лабораторные работы по обеспечению безопасности IT-инфраструктуры

## Лабораторная работа 1: Инвентаризация и сетевое сегментирование
* **Цель:** Выполнить инвентаризацию существующих ресурсов и настроить сетевое сегментирование.
* **Задачи:**
  * Использовать Ansible для автоматизации инвентаризации серверов.
  * Настроить VLAN или подсети для сегментации сети.
  * Ограничить доступ между сегментами с помощью правил брандмауэра.

## Лабораторная работа 2: Настройка мониторинга
* **Цель:** Настроить систему мониторинга для отслеживания состояния инфраструктуры.
* **Задачи:**
  * Установить и настроить Prometheus для сбора метрик с серверов.
  * Настроить Grafana для визуализации собранных данных.
  * Создать дашборды для мониторинга ключевых показателей производительности.

## Лабораторная работа 3: Настройка SIEM
* **Цель:** Установить и настроить систему управления информацией и событиями безопасности (SIEM).
* **Задачи:**
  * Установить Wazooh и интегрировать его с логами серверов.
  * Настроить правила для обнаружения подозрительных действий.
  * Провести симуляцию инцидента и проанализировать его в SIEM.

## Лабораторная работа 4: Настройка межсетевого экрана и VPN
* **Цель:** Настроить межсетевой экран и VPN для безопасного доступа.
* **Задачи:**
  * Настроить iptables или UFW для управления входящим и исходящим трафиком.
  * Установить и настроить OpenVPN для безопасного доступа к ресурсам.
  * Провести тестирование доступа через VPN.

## Лабораторная работа 5: Управление уязвимостями
* **Цель:** Провести оценку уязвимостей инфраструктуры.
* **Задачи:**
  * Использовать OpenVAS для сканирования серверов на наличие уязвимостей.
  * Анализировать результаты сканирования и составить отчет.
  * Устранить выявленные уязвимости и повторно провести сканирование.

## Лабораторная работа 6: Настройка шифрования данных
* **Цель:** Реализовать шифрование данных на уровне файловой системы и трафика.
* **Задачи:**
  * Настроить LUKS для шифрования дисков на Ubuntu.
  * Настроить TLS/SSL для защиты трафика между серверами.
  * Провести тестирование на доступность зашифрованных данных.

## Лабораторная работа 7: Настройка контроля доступа и аутентификации
* **Цель:** Внедрить многофакторную аутентификацию и управление доступом.
* **Задачи:**
  * Настроить LDAP или Active Directory для управления пользователями.
  * Реализовать многофакторную аутентификацию для доступа к критическим ресурсам.
  * Провести аудит прав доступа пользователей.

## Лабораторная работа 8: Реакция на инциденты и восстановление после сбоев
* **Цель:** Разработать план реагирования на инциденты и восстановление инфраструктуры.
* **Задачи:**
  * Создать план реагирования на инциденты и провести его обсуждение.
  * Настроить резервное копирование данных и конфигураций.
  * Провести тестирование восстановления системы после сбоя.
